RhinoSpider Connection Test

Connection Test Tool

This tool tests connections to the RhinoSpider proxy servers using all available methods:

  • HTTPS with domain name (preferred for Chrome Store compliance)
  • HTTP with domain name (fallback if HTTPS fails)
  • Direct IP connection (last resort fallback)

Connection Troubleshooting

If you're experiencing connection issues, try the following:

  1. Check if your DNS is resolving the domains correctly
  2. Verify that the proxy servers are running on the Digital Ocean server
  3. Check if the SSL certificates are valid and properly installed
  4. Ensure that the nginx configuration is correct
  5. Verify that the CORS headers are properly set

For more information, see the HTTPS Fix README.
